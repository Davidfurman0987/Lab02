# 4 Легенди UFC

Проект, присвячений чотирьом найвидатнішим бійцям в історії Ultimate Fighting Championship.

## Бійці

1. Джон "Bones" Джонс - Наймолодший чемпіон в історії UFC
2. Джордж "Rush" Сент-П'єр - Дворазовий чемпіон у напівсередній вазі
3. Андерсон "The Spider" Сільва - Найдовше правлячий чемпіон
4. Нік "The Diamond" Діаз - Один з найхаризматичніших бійців

## Технології

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Семантична розмітка
- Адаптивний дизайн
- Font Awesome іконки
- Google Fonts
